# Description
Due to immense dissastifaction with the current map at Carleton University we at High-Tier Tech decided to overhaul the Carleton Campus Map so that it is more intuitive and accessible than the current on being displayed. We hope to accomplish 3 things:

- Users have an easier time navigating the campus
- They can reach their desired destinations quicker
- Users understand the multiple ways to traverse the campus from key locations
